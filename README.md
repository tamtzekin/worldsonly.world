# Worlds Only - Band Landing Page

A minimalist landing page for Worlds Only featuring interactive artwork and custom cursor effects.

## Features

- Interactive custom cursor with white glow effect
- Responsive artwork display
- Clean, modern design
- Optimized for web deployment

## Files

- `index.html` - Main landing page
- `cursor.js` - Custom cursor functionality
- `artwork.png` - Band artwork
- `.gitignore` - Git ignore rules

## Deployment

### For Hostinger

1. Upload all files to your domain's `public_html` folder
2. Ensure `index.html` is in the root directory
3. Files should be accessible at `https://worldsonly.world`

### File Structure
```
public_html/
├── index.html
├── cursor.js
├── artwork.png
└── .gitignore
```

## Browser Compatibility

- Modern browsers with CSS3 support
- JavaScript enabled required for cursor effects
